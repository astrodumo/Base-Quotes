## BASE QUOTES
A beautiful, AI-powered web application that generates personalized philosophical quotes based on your current mood. Whether you're feeling reflective, anxious, joyful, or lost, Philosophical Compass offers timeless wisdom to guide you.

## Features

- 8 Mood Categories- Reflective, Anxious, Joyful, Sad, Inspired, Lost, Grateful, and Angry
- AI-Powered Quotes - Personalized philosophical wisdom generated by Claude AI
- Curated Collection- 240 handpicked quotes from history's greatest philosophers
- Beautiful UI - Mood-based gradient themes with smooth animations
- Responsive Design - Works seamlessly on desktop, tablet, and mobile
- Lightning Fast - Built with Vite for optimal performance


##  Tech Stack

- **Frontend:** React 18 + Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **AI:** Anthropic Claude API (Sonnet 4)
- **Deployment:** Vercel

Installation
- Node.js 16+ and npm

### Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/BASE-QUOTES.git
   cd philosophical-compass
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173`

### Using AI-Generated Quotes

To enable AI-powered quote generation:

1. **Get an Anthropic API key:**
   - Sign up at [console.anthropic.com](https://console.anthropic.com/)
   - Create an API key

2. **Set up environment variable:**
   - Create a `.env` file in your project root:
     ```
     ANTHROPIC_API_KEY=your_api_key_here
     ```

3. **Deploy with Vercel:**
   - Add the environment variable in Vercel dashboard
   - The serverless API route will handle quote generation
